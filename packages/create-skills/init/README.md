# init

Odoo skills project for AI agent integration.

## Setup

1. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```

2. Edit `.env` with your Odoo credentials:
   ```
   ODOO_URL=https://your-odoo-instance.com
   ODOO_DB=your_database
   ODOO_USER=your_username
   ODOO_PASSWORD=your_password
   ```

## Structure

- `SKILL.md` - Main skill router (entry point for AI agents)
- `AGENTS.md` - Instructions for AI agents
- `base/` - Core Odoo knowledge modules
- `skills/` - Instance-specific skills (you create these)

## Usage

### With Claude Code

Open this directory in Claude Code. The SKILL.md will be automatically loaded.

Ask Claude to:
- "Connect to Odoo and list available models"
- "Introspect the crm.lead model"
- "Generate a skill for creating CRM leads"

### With Other Agents

Instruct your agent to read `SKILL.md` for available capabilities.

## Creating Instance Skills

1. Ask your agent to introspect the target model
2. Generate a skill in `skills/{model-action}.md`
3. Update `SKILL.md` to reference the new skill

## Commands

```bash
# Validate skills and check references
```
