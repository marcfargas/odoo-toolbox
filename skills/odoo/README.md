# Odoo Skills

Odoo knowledge modules for AI agents. Start with `SKILL.md`.

## Structure

- `SKILL.md` — Entry point and router (read this first)
- `base/` — Core knowledge: connection, CRUD, domains, field types, introspection, modules, properties, search
- `mail/` — Messaging: chatter, activities, discuss channels
- `modules/` — Module-specific: accounting, timesheets
- `oca/` — OCA modules: MIS Builder

## Environment Variables

```bash
ODOO_URL=http://localhost:8069
ODOO_DB=my_database
ODOO_USER=admin
ODOO_PASSWORD=my_secure_password
```
